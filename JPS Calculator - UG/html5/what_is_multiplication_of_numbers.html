What is Multiplication of Numbers

Multiplication is a type mathematical calculation used to find the total quantity of an object when they are arranged in groups of equal quantity. Instead of adding the same number repeatedly, you can use multiplication for a faster and easier calculation. * or × are the symbols for multiplication.

If you want to multiply two numbers, write the calculation as "1st number" × "2nd number" = "result". Using this process, you find the total number of objects available in two groups. Refer to Multiplication Table to see the tables from 1 through 25.

Suppose you have three Rs 2 coins and you want to know what is the total amount of you have. Then, you have to multiply 2 and 3 (as there are 3 groups of Rs 2). So you should follow the below process:
  1. Write the numbers in a single line with × in between them; 3 × 2 = ?
  2. Add 2 three times; 2 + 2 + 2
  3. That equals 6.
Note:
Hence, if you remember 3 × 2 = 6, it makes your calculation easier.

So, the result is "6".

What is Division of Numbers

Division is a type of mathmatical calculation which is used to create groups of an object such that, each group has the same quantity of the object. / or ÷ are the symbols for division.

If you want to divide a number, write the calculation as "1st number" ÷ "2nd number" = "result". Using this process, you find the number of objects in each group and the number of it which can not be devided equally.
  • Dividend : 1st number (The total number to be divided).
  • Divisor : 2nd number (The number of groups).
  • Quotient : Result (The number of items in each group)
  • Reminder : The amount left over object (if the number cannot be divided equally)
Note:
The dividend is always greater than the divisor in the process of division.
Suppose you have 6 choclates in hand and you want to give them to 2 of your friends, so that your friends get the same number of chocolates. Then, you have to divide 6 by 2 (as you create 2 groups out of 6 chocolates). So you should follow the below process:
  1. Write the numbers in a single line with ÷ in between them; 6 ÷ 2 = ?
  2. Give one chocolate to each of your friends. Do the same till you have no chocolate with you.
  3. You can see that there are 3 chocolates with each of your friends.

So, the result is "3".
